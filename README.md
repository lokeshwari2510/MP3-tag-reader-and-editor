# MP3 Tag Reader and Editor

## Description
A C program to read and edit ID3v2 MP3 tags.

## Features
- View MP3 tags
- Edit Title
- Edit Artist
- Edit Album
- Edit Year
- Edit Genre
- Edit Comment

## Commands

View tags:

./a.out -r sample.mp3

Edit title:

./a.out -e -t "New Title" sample.mp3

Help:

./a.out -h

## Technologies
- C Language
- File Handling
- ID3v2 MP3 Metadata
